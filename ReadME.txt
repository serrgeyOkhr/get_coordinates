Приложение для определения точных координат места с помощью сервиса Dadata.

Перед запуском убедитесь, что у вас установленны необходимые зависимости (requirements.txt)

При первом запуске обязательно перейдите в настройки и задайте ваш API key.
Для получения API ключа и секретного ключа Вам необходимо зарегистрироваться в сервисе, после чего необходимые данные будут доступны в Вашем личном кабинете по ссылке: https://dadata.ru/profile/#info
